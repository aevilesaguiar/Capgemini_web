Hospedagem e Servidores

Relembrando como a Internet funciona, nós temos um computador esse computador se conecta através do protocolo TCP ao servidor através do http os documentos que ele quer consumir do servidor. Um servidor pode ser qualquer computador, desde que você registre o seu computador num DNS.Mas hospedar num servidor local não é vantajoso, pois é necessário ter um link de alta qualidade, alta disponibilidade, senão as pessoas não conseguirão acessar seu site, por isso temos os DataCenters, que são administradas as centrais de dados, onde tem controle de temperatura, controle de acesso e alta disponibilidade

Esse serviço feito por uma empresa que tem um DataCenter se chama hospedagem, então quando voce paga um datacenter , você está pagando por todo o serviço de manutenção. Hospedagem é um serviço prestado por uma empresa ou um indivíduo e esse serviço é feito através de servidores e esses servidores quando estão todos juntos é chamado de DataCenters

Diferença entre conteúdo Estático e Dinâmico

Os sites estáticos são fixos e exibem o mesmo conteúdo para todos os usuários, geralmente escritos exclusivamente em HTML, mas podem ser aúdios, arquivos, mas que eles existem no servidor independente da solicitação do cliente. Um site dinâmico , por outro lado, é aquele que pode exibir conteúdo diferente e fornecer interação com o usuário, utilizando programação avançada e bancos de dados, além do HTML.

Servidor WEB - Como funciona?

Um servidor web não conversa direto com o Banco de dados, ele não executa arquivos php direto, ele apenas entrega conteúdo pronto para o cliente no nosso caso o Navegador. Se eu entrego conteúdo dinamico ele chama o Servidor de aplicação e o servidor de aplicação entrega para o servidor web o conteudo, que pode ser consumido pela conexão. O servidor web seria como se fosse um entregador de conteúdo pronto para ser consumido.
