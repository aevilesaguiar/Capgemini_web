Acessibilidade (Mozilla Foundation)

  • - Visa aumentar a facilidade de acesso ao seu site ou software web.
  • - Precisa pensar de forma que todos possam acessar o conteúdo.

Acessibilidade tem a ver com inclusão;

Aplicações acessiveis independente de capacidade auditivas, visuais fisicas ou cognitivas.

Deve ser acessivel por:

  • - Teclado
  • - Mouse
  • - Touch Screen
  • - Comandos de voz
  • - Existem muitos outros...

Sites seguros do ponto de vista de saúde, exemplos:

  • - Sem cores não pensadas
  • - Sem telas piscantes que prejudicam epiléticos
  • - Sem itens que podem causar dores de cabeça

Acessibilidade Móvel: Entende-se por acessibilidade Web móvel a inclusão de usuários com as mais diversas especificidades, para o acesso de conteúdo da Web através de dispositivos móveis (smartphones, tablets, etc.).

Para que serve o TalkBack?
Ao ativar o TalkBack, você ouvirá uma resposta do dispositivo após realizar algumas ações, como alternar entre as páginas da tela principal ou criar uma pasta na tela principal. O erro afeta todos os aplicativos que utilizam serviços de acessibilidade.

É muito importante testar em diversos dispositivos o layout do seu website ou aplicativo. Verificar se os menus são acessiveis.

CSS e javaScript no âmbito da Acessibilidade

JavaScript e CSS tem a mesma importancia que o html. Pois eles podem causar danos a acessibilidade.

As Vantagens de um site acessivel e com boa semântica podem ser:

  • - Bom para SEO
  • - Boa usabilidade em dispositivos móveis
  • - Fácil de ser desenvolvida
  • - Aumenta a audiencia do site e com isso os ganhos
