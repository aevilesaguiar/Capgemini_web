Estrutura Básica HTML

HTML é toda baseada em tags.

Tags: são marcações, elas classificam o conteúdo. Elas são ignoradas pelo o usuario, mas serão utilizadas para renderizar o conteúdo. As tags podem ter atributos.

No html5 as tags passam a ser estruturadas semanticamentes.

SGML:SGML significa Linguagem de marcação generalizada padrão é uma linguagem de marcação generalizada padrão que faz uso de um superconjunto de linguagens de marcação amplamente utilizadas, como HTML e XML. É utilizado para marcar arquivos e tem a vantagem de não depender mais de um determinado aplicativo.
