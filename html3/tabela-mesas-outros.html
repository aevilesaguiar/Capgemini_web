Tabelas Mesas e Outros

O elemento HTML Table ("table") representa dados em duas dimensões ou mais.Uma tabela é dividida em linhas (com a tag "tr"), e cada linha é dividida em células de dados (com a tag "td"). As letras td significam "table data," que é o conteúdo de uma célula de dados. Uma célula de dados pode conter texto, imagens, listas, parágrafos, formulários, filetes horizontais, tabelas, etc.
