Componentes do formulário: input

O input tem vários tipos, e isso se refere principalmente ao tipo de entrada de dados que será feito. qual o meio de entrada de dados, exemplo teclado, botão...

name : é como será identificado a nossa variável que será enviada ao servidor

value é o valor inicial

placeholder: vai aparecer quando o usuario não digitar nada no campo.

Campos html5

  • type="email"
  • type="password"
  • type="hiden" a informação do campo fica escondido
  • type="search" muito usado para campos de buscas, quando queremos alterar a querie
  • type="tel"
  • type="number" aceita apenas numeros
  • type="range" define um controle para inserir um número cujo valor exato não é importante (como um controle deslizante)
  • type="checkbox" selecionar varios itens dependendo do que o usuario seleciona
  • type="radio" selecionar apenas 1 item
  • textarea:O elemento HTML
