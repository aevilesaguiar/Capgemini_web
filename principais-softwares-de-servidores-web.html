Principais softwares de servidores web (WEB SERVER)

  • Apache: responsavel por 50% a 60% de uso na internet. Software Livre, surgiu em 1995, foi o primeiro servidor gratuito que permitia a qualquer pessoa lançar um website na internet. A fundação Apache é responsável pelo o Apache. O apache web server trabalha com módulos. Ele é muito facil de configurar. Além disso tem um painel de controle CPanel que administra serviços de email, administra também outros serviços de hospedagem.
  • NGINX: esse servidor é um pouco mais complicado de configurar, mas tem mais performance . Não é indicado para iniciantes. trabalha com mais conexões simultâneamente
  • IIS: IIS é um servidor web criado pela Microsoft para seus sistemas operacionais para servidores. Sua primeira versão foi introduzida com o Windows NT Server versão 4, e passou por várias atualizações. É também o sucessor do PWS. A versão mais recente é o IIS 10.
  • Litespeed: parecido com o Apache, trás muito mais performance
  • NODE JS: node + express ele não é exatamente um web server, ele é um servidor web para JavaScript, usa o mesmo renderizador do JavaScript se usar o express você consegue criar um servidor web. Normalmente o Node é usado com NGINX, utiliza-se o NGINX na frente como um proxy. Ele não é um servidor web e sim uma forma de entregar conteudo através de uma porta web, através do proprio ngnix servindo como proxy para esse conteúdo.
