Tipo de arquivo de imagem e guia de formato

Os formatos de arquivo de imagem mais usados ​​na web estão listados abaixo.

Tipos de arquivo de imagem comuns
Abreviação Formato de arquivo Tipo MIME Extensões de arquivo) Resumo
APNG Gráficos de rede portátil animados image/apng .apng Boa escolha para sequências de animação sem perdas (o GIF tem menos desempenho). AVIF e WebP têm melhor desempenho, mas suporte menos amplo ao navegador. Suportado : Chrome, Edge, Firefox, Opera, Safari.
AVIF Formato de arquivo de imagem AV1 image/avif .avif Boa escolha para imagens e imagens animadas devido ao alto desempenho e formato de imagem livre de royalties. Ele oferece uma compactação muito melhor do que PNG ou JPEG com suporte para profundidades de cor mais altas, quadros animados, transparência etc. Observe que ao usar AVIF, você deve incluir fallbacks para formatos com melhor suporte ao navegador (ou seja, usando o elemento). Suportado: Chrome, Opera, Firefox (apenas imagens fixas: imagens animadas não implementadas).
GIF Formato de intercâmbio de gráficos image/gif .gif Boa escolha para imagens e animações simples. Prefira PNG para imagens estáticas sem perdas e indexadas e considere WebP, AVIF ou APNG para sequências de animação. Suportado: Chrome, Edge, Firefox, IE, Opera, Safari.
JPEG Imagem do Grupo Conjunto de Especialistas em Fotografia image/jpeg .jpg, .jpeg, .jfif, .pjpeg,.pjp Boa escolha para compactação com perdas de imagens estáticas (atualmente a mais popular). Prefira PNG quando for necessária uma reprodução mais precisa da imagem, ou WebP/AVIF se for necessária uma melhor reprodução e maior compactação. Suporte: Chrome, Edge, Firefox, IE, Opera, Safari.
PNG Gráficos Portáteis de Rede image/png .png O PNG é preferível ao JPEG para uma reprodução mais precisa das imagens de origem ou quando a transparência é necessária. O WebP/AVIF oferece compactação e reprodução ainda melhores, mas o suporte ao navegador é mais limitado. Suporte: Chrome, Edge, Firefox, IE, Opera, Safari.
SVG Gráficos vetoriais escalonáveis image/svg+xml .svg Formato de imagem vetorial; ideal para elementos de interface de usuário, ícones, diagramas, etc., que devem ser desenhados com precisão em diferentes tamanhos. Suporte: Chrome, Edge, Firefox, IE, Opera, Safari.
WebP Formato de imagem da Web image/webp .webp Excelente escolha para imagens e imagens animadas. WebP oferece uma compactação muito melhor do que PNG ou JPEG com suporte para profundidades de cor mais altas, quadros animados, transparência etc. AVIF oferece compactação um pouco melhor, mas não é tão bem suportado em navegadores e não suporta renderização progressiva. Suporte: Chrome, Edge, Firefox, Opera, Safari
